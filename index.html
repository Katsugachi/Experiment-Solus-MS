About Solus MS

Cloud Gaming, Redefined.

Solus MS now features built-in CloudMoon streaming. Sign in with Google to launch Android games directly in the cloud — no installs, no downloads, no hardware limits. Built by katsugachi.

Version 2.2.0 · Cloud Edition

Cloud Streaming

Real Android instances via CloudMoon. Play mobile games at full fidelity, right in your browser.

Google Sign-In

One-click auth links your Google account to a live CloudMoon session instantly.

Stealth Features

Tab cloak, invisibility cloak, panic key, and reflection cloaking — all built in.

Solus MS is a custom browser emulator launching interface designed by katsugachi on GitHub.
Features include tab cloaking to disguise as any tab easily. Made completely in HTML and has reflection cloaking to avoid being seen even from direct view at an angle
