About Solus MS

☁ Cloud Gaming, Redefined.

Solus MS features built-in CloudMoon streaming — fully self-hosted on your own GitHub Pages. Sign in to launch Android games in the cloud with no time limits, no ads, no installs. Built by katsugachi.

Version 2.3.0 · Self-Hosted Cloud Edition
☁

Self-Hosted Streaming

CloudMoon streams now run through your GitHub Pages run-site — no redirects to external sites.

∞

No Time Limits

The 15-minute ad timer has been removed. Play as long as your CloudMoon session allows.

👁

Stealth Features

Tab cloak, invisibility cloak, panic key, and reflection cloaking — all built in.

Solus MS is a custom browser emulator / game launching interface. Made completely in HTML.
